# Stow Setup

Geridos com **GNU Stow**.

## Como instalar

1. Clone: `git clone https://github.com/MiguelGrilo/dotfiles.git ~/dotfiles`
2. Instale o Stow: `sudo pacman -S stow`
3. Aplique as configs: `cd ~/dotfiles && stow *`" > README.md

## Enviar o README para o GitHub

```
git add README.md
git commit -m "Adicionado README informativo"
git push origin main
```
